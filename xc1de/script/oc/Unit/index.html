    Unit OC/Object¶

    Represents a battle unit object.

    Constructor¶

    oc unit(string bdat, int id = 0, int unk = 0)

    Gets a bdat unit.

    Valid names:

    • FLD_npclist or npc
    • BTL_enelist or ene
    • BTL_pclist or pc
    • player
